严选福礼

 

杭银发放福粒

发放福粒

基本信息

Path： /bankHang/sendWelfareV2

Method： POST

接口描述：

请求参数

Headers

名称 类型 是否必须 默认值 备注 其他信息
enterpriseNo string 必须 渠道ID
type number 必须 发放类型【1指定账户】暂只支持这一种
actName string 必须 批次名称（不能多于20个字）
bankHangActNum String 必须 行方活动号

用于福粒变动明细查询

actendTime number 必须 回收时间

限定时间回收 批次创建时间+6个月

userInfos object [] 必须 用户信息

item 类型: object
├─ phone number 必须 手机号
├─ sendAmount number 必须 发放金额（1、传的是福粒数量，不是福粒金额；2、大于0小于50000的正数，支持到小数点后两位）(传10即10福粒)
sign string 必须 加密后数据 加密信息去除actendTime，增加bankHangActNum

 

返回数据

名称 类型 是否必须 默认值 备注 其他信息
code number 必须 200成功，其他异常
msg string 非必须 异常信息
data object 必须
├─ state number 必须 发放状态（1全部发放成功；2部分失败；3全部失败）
├─ errorState number 必须 批次信息发放失败原因状态(1发放类型不存在；2渠道id不存在；3回收时间不符合要求；4批次名称不符合要求；5行方活动号为空)
├─ successPhone string [] 必须 发放成功的手机号

item 类型: string
├─ 非必须 手机号
├─ errorPhone object [] 必须 用户信息失败详情

item 类型: object
├─ phone number 必须 手机号
├─ msg number 必须 失败原因(1手机号格式不正确；2手机号重复：重复的手机号都判定为失败；3福利金发放数不符合要求)

 

福粒变动明细查询（最长可查90天）

基本信息

Path： /bankHang/sendWelfareListAll

Method： POST

接口描述：

请求参数

Headers

返回数据

名称 类型 是否必须 默认值 备注 其他信息
code string 必须
msg string 非必须
data object 必须
├─ pageVo object 必须
├─ totalCount number 必须 总数
├─ totalPage number 必须 总页数
├─ page number 必须 页码
├─ pageSize number 必须 页大小
├─ welfareChangeDetail object [] 必须

item 类型: object
├─ createTime number 必须 福粒变动时间
├─ type number 必须 变动类型（1获得福粒；3消费订单；2退单返还）
├─ amount number 必须 变动数量
├─ sign number 必须 变更金额的符号（0扣减1增加）
├─ phone string 必须 手机号
├─ batchNumber object [] 必须 变动明细

item 类型: object
├─ batchName string 必须 批次名称
├─ bankHangActNum string 必须 行方活动号
├─ amount number 必须 批次变动数量
├─ sign number 必须 变更金额的符号（0扣减1增加）
